## gml2json ##
##### convert gml to json #####

#### Install ####
```shell
cd go2json
go build
```

#### Usage ####
```shell

Usage: gml2json [options...]

Options:
  -i  Input gml file. (default "example.gml")
  -o  Output json file. (default "convert.json")
  -c  Config file. (default "config.yaml")

Example:

  gml2json -i example.gml -o example.json -c config.yaml

```
